# Placeholder images

Add your own images here:

- `products/<product-slug>.png`
- `brands/<brand-slug>.png`

Slug format used by the app:
- lowercase
- spaces/special chars replaced with `-`

Examples:
- `Smartphone` -> `smartphone.png`
- `Apple` -> `apple.png`
- `HP` -> `hp.png`

Fallback files used when image is missing:
- `products/_placeholder.svg`
- `brands/_placeholder.svg`
